ES6 Workshop
============

Ziele

* selber ES6 lernen
* zeigen, dass ES6 besser ist als klassisches Javascript
* besseren Javascript Code schreiben (Namespaces, Struktur, Klassen, Re-use)


Themen die gezeigt/gelernt werden sollen

* Adoption
  * Browser Support
  * Transpiler

* 10 Kernfeatures, siehe https://webapplog.com/es6/
  * Default Parameters in ES6
  * Template Literals in ES6
  * Multi-line Strings in ES6
  * Destructuring Assignment in ES6
  * Enhanced Object Literals in ES6
  * Arrow Functions in ES6
  * Promises in ES6
  * Block-Scoped Constructs Let and Const
  * Classes in ES6
  * Modules in ES6

* Tech Stack
  * Node Project Setup (NPM)
    * `sudo npm install`
  * Webpack (pack, watch)
    * erstellt public assets
    * wie sind Module?
  * ES6 Features nach Liste durch Serie von kleinen Requirements
    * see !! comments
  * React (UI)
  * Test?
